## Day 1: Basic Bank Account Model

Implemented:

- abstract account model `AbstractAccount`
- concrete account class `BankAccount`
- account statuses via `AccountStatus`
- supported currencies via `Currency`
- custom domain exceptions
- deposit and withdrawal operations
- validation for amount, owner, status, and currency
- read-only balance property
- serializable `to_dict()` method for future analytics and reporting
- demo scenario in `main.py`
